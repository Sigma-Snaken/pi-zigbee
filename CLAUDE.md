# Project: sigma-button-controller

## Quick Start
Run `/sigma` at the start of each session to load domain context.

## Skills Used
- `kachaka:kachaka-sdk` — Robot control patterns
- `sigma-devkit:kachaka-app-architecture` — App architecture reference
- `sigma-devkit:design-tokens` — Frontend styling (Amber/Cream)
- `sigma-devkit:deploy` — Docker deployment
- `sigma-devkit:hil-testing` — Test infrastructure

## Project Context
- Type: Robot app (button-to-action controller)
- Framework: FastAPI (backend) + Vanilla JS (frontend)
- Database: SQLite
- Robot: Kachaka (multi-robot via RobotManager)
- Deployment: Docker Compose on Pi
- AI: None
